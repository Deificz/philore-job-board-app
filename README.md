**Philore Job Board**

**Description**

A sample job board that mocks online job listing where you can apply, track, and see if your current skills matches the available job

**Tools**

- Next.js ( Frontend Framework ) 
- TypeScript ( Language )
- shadcn ( UI Library )
- Tailwind ( CSS Framwork )
- json-server ( For mock REST API )

**Required Packages**

v24.6.0 or up - node version

**Initialization Commands**

1. Run **npm start**
2. Run **npm run json-server**
3. Open web-app on **http://localhost:300/** in your browser
